Littlegods Mod Menu is one of the most complete projects created for Call of Duty: Black Ops 2 Zombies (Plutonium).
It includes a visual interface through a redesigned menu where you can use multiple GSC functions for the player.
The menu is designed to be modular, optimized, and completely free for the Plutonium community.
You are free to edit it your way and use it in your games.

## üöÄ INSTALLATION

1.  Download the ZIP file of the mod
2.  Extract the files into your Plutonium folder: `storage/t6/raw/scripts/zm/`
3.  Make sure the files are in the correct path
4.  Start the game and join a Zombies match

## ‚ö†Ô∏è IMPORTANT NOTES

### **Restrictions and Security:**
-   **God Mode required**: Round system and some functions require God Mode activated
-   **Developer Mode locked by default**: Requires a password to access the full menu
-   **Stats protection**: Developer Mode disables record saving to prevent cheating
-   **Mod conflicts**: Some mods have mutual restrictions (UFO + Forge, Aimbot + Forge, etc.)

### **Technical Considerations:**
-   **Performance**: Using many mods simultaneously can affect performance
-   **Backup recommended**: Back up original files before installing
-   **Config persistence**: All preferences are saved automatically
-   **Technical limits**: Max 255 rounds, 99999 health, etc.

### **Developer Mode System:**
-   **Restricted access**: Full menu locked by default
-   **Password required**: Authentication system for advanced mods
-   **Safe mode**: Preserves the integrity of legitimate stats
-   **Final effects**: Some effects are disabled when developer mode is active
